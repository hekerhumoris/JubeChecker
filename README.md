# Welcome in Jubelio Checker Tools

You can using this tools for checking :

- HTTPS/SSL
- CSP (Content Security Policy)
  - Clickjacking
  - XSS
  - More..
- X-Frame Options
- Directory Listing
- More Coming Soon..

How to Install :

- `git clone https://github.com/hekerhumoris/JubeChecker.git`
- `cd JubeChecker`
- `chmod +x setup.sh`
- `./setup.sh`
- run tools `python3 jube.py`
